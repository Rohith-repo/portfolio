# Rohith's Portfolio

A simple single-page portfolio showcasing projects, skills and a playful terminal-like UI.

Features
- Static site built with HTML, CSS and vanilla JavaScript
- Animated canvas background and scroll reveal effects
- Sections: Hero, About, Skills, Pipeline, Projects, GitHub, Research, Experience, Contact

Quick start
1. Open [index.html](index.html) in your browser (no build step required).

Customize
- Edit the `CONFIG` object near the bottom of [index.html](index.html) to change `name`, `githubUsername`, `email`, and `resumeUrl`.
- Styles are currently embedded in [index.html](index.html) and also a `styles.css` file exists (work in progress).
- JavaScript is embedded in [index.html](index.html) under the `<script>` block.

Notes
- The site uses the Google font `Press Start 2P` for a pixel aesthetic.
- To fully externalize CSS/JS, link `styles.css` and `scripts.js` from [index.html](index.html) (TODO in repo).
